# BOS Page Style

本目录收录 BOS 系统的页面样式参考，作为 BOS 设计 skill 的一部分。

## 用途

当 AI 根据需求设计页面时，参考此目录下的 page style 文件，生成符合规范的页面布局和样式。

## 目录结构

```
BOS-page-style/
├── README.md        # 本文件，维护说明
├── index.md         # 结构化索引，供 AI 检索
└── ...              # 各 page style 文件
```

## 如何新增 Page Style

1. 在本目录下新建 `.md` 文件，命名建议：`<页面类型>.md`（如 `list-page.md`、`detail-page.md`）
2. 在 `index.md` 中添加对应条目
3. 文件内容应包含：页面适用场景、布局结构、关键样式规则、示例代码（如有）
