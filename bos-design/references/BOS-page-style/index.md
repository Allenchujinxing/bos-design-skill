# BOS Page Style Index

> 供 AI 设计 skill 检索使用。Agent 分析完需求后，对照「匹配意图」列判断该参考哪个 page style 文件。

## 页面样式列表

| 页面类型 | 文件 | 当需求中出现这些意图时参考 |
|----------|------|--------------------------|
| 区域选择（Area Selection） | area-selection.md | 选择城市/街道/网格、切换工作范围、层级筛选定位、选择业务线 |
| 对象搜索（Object Search） | object-search.md | 搜索查找特定对象（人/车/站/城/计划/任务）、跳转到独立搜索页面 |
| 页内辅助操作（Inline Action） | inline-action.md | 在当前页面内完成辅助操作：筛选条件、排序、选日期时间、选择选项、展示功能入口、查看详情、拍照上报 |
| 信息录入（Data Entry） | data-entry.md | 创建计划、配置规则、指派任务、修改指派、填写提交信息、批量操作配置 |
| 经营看板（Operation Overview） | operation-overview.md | 查看运营数据、实时指标、趋势对比、经营分析，核心是看数据做判断 |
| 内容明细（Content Detail） | content-detail.md | 浏览和操作一组业务对象（任务、站点、网格、街道、派单计划），带筛选排序和卡片操作 |
| 信息卡片范式（Card Patterns） | card-patterns.md | 需要展示业务对象信息时（任务、车辆、人员、站点、订单、计划），参考对应对象的卡片信息组织范式 |
