# Dropdown Panel 下拉面板

## Source

- Figma file: `Hello-TaroUI-Kit-2026`
- Component page: `14539:4276`、`15442:5223`

## Purpose

筛选栏点击后展开的内容面板，承载筛选选项的具体内容。配合 Filter Bar 筛选栏使用。

与 `input/filter-bar.md` 组合使用。筛选栏负责触发和状态展示，下拉面板负责内容和选择交互。实现时必须同时引用两个文件。

1. 从筛选栏下方向下展开，展开时背景出现半透明遮罩
2. 支持单选（无底部按钮，点击即关闭）和多选（必须有确认按钮）
3. 面板底部圆角 12px，顶部无圆角（紧贴筛选栏）
4. 支持 7 种内容布局模式

This file records the Figma visual style only.

## 视觉 Token 声明

本文件中的颜色、圆角、间距等视觉属性引用自搭配的 Design 规范文件（如 `references/design.md`）。当切换到其他视觉系统时（如 HelloRide、HelloSaaS），对应 token 的色值会变化，组件结构和行为不变。

## 模式 4：双列网格模式

左侧分类导航 + 右侧标签网格。适用于多主题/多分类 + 标签选择的场景。

### 整体布局

| Property | Value |

### 左侧分类栏

| Property | Value |
|---|---|
| 宽度 | 92px |
| 背景 | gray7 `#F0F3F5` |
| 分类项高度 | 46px |
| 分类项内边距 | 0 12px |
| 文字（默认） | 14px / regular 400 / gray1 `#111111` / line-height 22px |
| 文字（选中） | 14px / semibold 600 / Branding-1 `#0076FF` |
| 选中态背景 | white1 `#FFFFFF` |
| 已选标记 | 左侧 4px 处，4×4px 圆点，Branding-1 `#0076FF`（多选场景下表示该分类有已选项） |

### 右侧内容区

| Property | Value |
|---|---|
| 宽度 | 剩余空间（390 - 92 = 298px） |
| 背景 | white1 `#FFFFFF` |
| 内边距 | 12px |
| 内容 | 标签网格（同模式 3 的网格项样式） |

### 右侧内容区标题

每个分类对应的内容区顶部有标题文字：

| Property | Value |
|---|---|
| 文字（默认） | 14px / semibold 600 / gray1 `#111111` / line-height 22px |
| 文字（高亮） | 14px / semibold 600 / Branding-1 `#0076FF` / line-height 22px |
| 下边距 | 8px |

标题高亮规则：当内容区滚动到该分类的区域时，对应标题变为蓝色；滚出该区域后恢复默认色。与左侧分类高亮联动一致。

### 滚动联动交互

- 右侧内容区**一次性展示所有分类的内容**（每个分类一个标题 + 对应内容），可上下滚动
- 右侧内容区上下滑动时，滚动到哪个分类的内容区域，左侧对应的分类项自动高亮
- 点击左侧分类项，右侧内容区平滑滚动（`scrollIntoView smooth`）到对应分类的锚点位置
- 点击左侧触发滚动时，需要 400ms 防抖，期间不响应滚动联动事件，避免左侧高亮闪烁
- 类似"锚点导航"交互模式

### 左侧蓝点标记规则

蓝点表示该分类下有已选项，是动态的：
- 当某个分类下有选中项时，该分类左侧出现蓝点
- 当该分类下所有选中项被取消时，蓝点消失
- 重置操作清空所有选中项时，所有蓝点消失
- 蓝点在多选场景下必须展示，帮助用户识别哪些分类有已选内容

### 筛选项状态联动

确认关闭面板后，对应的筛选项需要体现已选状态：
- 有选中项时：筛选项文字加粗变蓝（Branding-1）
- 无选中项时：筛选项恢复默认态（regular gray1）
- 重置后：筛选项恢复默认态

## 交互模式

### 面板互斥

同一时间只能有一个面板处于展开状态。切换筛选项时：
- 当前已打开的面板**直接关闭（无收起动画）**
- 新面板正常播放展开动画
- 遮罩保持显示，不闪烁

### 主动关闭

以下操作触发面板收起（有收起动画）：
- 点击遮罩
- 点击确认按钮
- 再次点击当前已激活的筛选项

### 遮罩范围

- 遮罩只覆盖筛选栏以下区域（`top` 从筛选栏底部开始）
- 筛选栏和导航栏不被遮罩覆盖，始终可见可操作

### 筛选项状态联动

| 状态 | 文字 | 箭头方向 | 箭头颜色 |
|---|---|---|---|
| 默认（未展开、无已选） | regular 400 / gray1 | 朝下 | gray3 `#879099` |
| 激活（面板展开中） | semibold 600 / Branding-1 | 朝上（旋转 180°） | Branding-1 `#0076FF` |
| 已选（面板关闭、有已选项） | semibold 600 / Branding-1 | 朝下 | gray3 `#879099` |

### 单选 vs 多选

| 模式 | 底部按钮 | 关闭方式 |
|---|---|---|
| 单选 | 无 | 点击选项即选中并关闭面板（延迟 120ms 后关闭） |
| 多选 | 重置 + 确认 | 点击确认关闭，点击选项仅切换选中态不关闭 |

### 按钮交互

| 按钮 | 点击行为 | pressed 态 |
|---|---|---|
| 确认 | 关闭面板（有收起动画） | 背景变为 Branding-0 `#0061D1` |
| 重置 | 清空当前面板所有选中项，面板保持展开 | 背景变为 Branding-2 `#C2DEFF` |

## Do Not Infer

- 不要在单选场景加底部操作栏 — 单选点击即关闭。
- 不要在多选场景省略底部操作栏 — 多选必须有确认按钮。
- 不要让面板顶部有圆角 — 顶部紧贴筛选栏，只有底部 12px 圆角。
- 不要省略遮罩 — 面板展开时必须有半透明遮罩（气泡菜单除外）。
- 不要让网格项超过三列 — 固定三列布局。
- 不要让列表项高度小于规定值 — 单行 46/56px，双行 68px。
- 不要在分组标题中使用 regular 字重 — 分组标题必须 semibold 600。
- 不要省略左侧分类的选中效果 — 选中项必须是白色背景 + 蓝色加粗文字。
- 不要省略滚动联动 — 双列网格模式右侧滚动必须联动左侧高亮。
- 不要在"不限"二级选项时展示三级列 — 三列模式的特殊规则。
- 不要让同一行标签高度不一致 — 网格模式中同行标签必须等高。
