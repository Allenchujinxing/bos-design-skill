# Loading 加载

## Source

- Figma file: `Hello-TaroUI-Kit-2026`
- Component page: `8373:12693`（全页面加载）、`8373:11294`（局部加载）

## Purpose

数据加载过程中的视觉反馈，告知用户当前正在处理中。分为全页面加载、局部模块加载和按钮加载三种场景。

This file records the Figma visual style only.

## 视觉 Token 声明

本文件中的颜色、圆角、间距等视觉属性引用自搭配的 Design 规范文件（如 `references/design.md`）。当切换到其他视觉系统时（如 HelloRide、HelloSaaS），对应 token 的色值会变化，组件结构和行为不变。

## 1. 全页面加载

页面中无其他内容，在页面中部显示循环 loading 动画。

### Loading 图标

| Property | Value |
|---|---|
| 图标 | `ic_loading_line`（循环旋转动画） |
| 尺寸 | 56×56px |
| 颜色 | Branding-1 |
| 动画 | 360° 持续旋转，1s 一圈，linear |
| 位置 | 页面水平居中，垂直居中 |

### 变体

| 变体 | 说明 |
|---|---|
| 纯图形 | 仅 loading 图标，无文字 |
| 图形 + 文字 | 图标下方 12px 间距 + 文字说明 |
| 图形 + 主副文字 | 图标下方 12px + 主文字 + 8px + 副文字 |

### 文字说明

| Property | Value |
|---|---|
| 主文字 | 16px / regular 400 / gray1 / line-height 24px / 居中 |
| 副文字 | 14px / regular 400 / gray3 / line-height 22px / 居中 |
| 图标与主文字间距 | 12px |
| 主文字与副文字间距 | 8px |

## 2. 前置加载（提交类页面）

提交类页面（如登录、订单提交）触发提交后，在当前页面显示加载过程。

### 容器

| Property | Value |
|---|---|
| 尺寸 | 自适应内容（最小 80×80px） |
| 背景 | white1 |
| 圆角 | `radius-xl` |
| 位置 | 页面水平居中，垂直居中 |

### 有遮罩

| Property | Value |
|---|---|
| 遮罩 | `rgba(0,0,0,0.7)` 全屏覆盖 |
| 容器投影 | 无（遮罩已提供层级区分） |

### 无遮罩

| Property | Value |
|---|---|
| 遮罩 | 无 |
| 容器投影 | 二级投影 `0 4px 8px 0 rgba(17,17,17,0.08)` |

### 加载过程类型

| 类型 | 说明 | 交互 |
|---|---|---|
| 不可逆 | 用户处于主流程必经场景 | 无法中断，等待完成 |
| 可逆 | 用户处于非主流程场景 | 允许用户手势回退或点击取消 |

## 3. 局部模块加载

页面已有内容，某个模块区域正在加载数据。

### 样式

| Property | Value |
|---|---|
| 图标 | `ic_loading_line` |
| 尺寸 | 24×24px 或 20×20px（根据模块大小） |
| 颜色 | gray3 |
| 动画 | 360° 持续旋转，1s 一圈，linear |
| 位置 | 模块区域水平居中 |
| 文字（可选） | 12px / regular 400 / gray3 / 图标下方 4px |

---

## 4. 按钮加载

按钮触发操作后，按钮内显示 loading 状态。

### 样式

| Property | Value |
|---|---|
| 图标 | `ic_loading_line` |
| 尺寸 | 与按钮内文字等高（16px / 14px / 12px，跟随按钮尺寸） |
| 颜色 | 跟随按钮文字颜色（primary 按钮为 anti 白色） |
| 动画 | 360° 持续旋转，1s 一圈，linear |
| 位置 | 替换按钮文字，或在文字左侧（gap 4px） |

### 按钮加载规则

- 加载中按钮保持原有宽度、高度、圆角不变
- 加载中按钮不可再次点击（disabled 态）
- 可选择隐藏文字只显示图标，或保留文字在图标右侧

---

## 动效

| 行为 | 说明 |
|---|---|
| 旋转 | 所有 loading 图标持续 360° 旋转，1s/圈，linear timing |
| 出现 | 淡入 opacity 0→1，duration 200ms |
| 消失 | 淡出 opacity 1→0，duration 200ms |

---

## Do Not Infer

- 不要省略旋转动画 — loading 图标必须持续旋转。
- 不要在全页面加载时显示页面内容 — 全页加载意味着内容未就绪。
- 不要让按钮加载时改变按钮尺寸 — 宽高保持稳定。
- 不要在无遮罩的前置加载中省略投影 — 投影是唯一的层级区分手段。
- 不要让 loading 图标静止 — 静止的图标无法传达"加载中"的含义。
