# Fab 浮动按钮

## Source

- Figma file: `Hello-TaroUI-Kit-2026`
- Component page: `8373:8435`
- Component set: `Floating Button 浮动按钮`

## Purpose

悬浮在屏幕上的操作按钮，代表界面中的重要操作，便于用户快捷操作。固定在屏幕某个位置，不随页面滚动。

This file records the Figma visual style only.

## 视觉 Token 声明

本文件中的颜色、圆角、间距等视觉属性引用自搭配的 Design 规范文件（如 `references/design.md`）。当切换到其他视觉系统时（如 HelloRide、HelloSaaS），对应 token 的色值会变化，组件结构和行为不变。

## 基础样式

| Property | Value |
|---|---|
| 形状 | 圆形 |
| 尺寸 | 40×40px |
| 背景 | Branding-1 |
| 投影 | 二级投影 `shadow-2` |
| 图标 | 20×20px，anti `white1` |
| 位置 | 固定定位，默认右下角 |
| 距右 | 12px |
| 距底 | 距底部安全区上方 12px（即 bottom: 34 + 12 = 46px） |

## 变体

| 变体 | 说明 |
|---|---|
| 图标按钮 | 圆形品牌色背景 + 白色图标 |
| 自定义内容 | 圆形容器内放置自定义图片/营销素材 |
| 带关闭按钮 | 右上角 18×18px 关闭图标，点击关闭浮动按钮 |

---

## 位置规则

| 位置 | 说明 |
|---|---|
| 右下角（默认） | 距右 12px，距底部安全区上方 12px |
| 右下角（有底部操作栏时） | 距底部操作栏上方 12px |
| 可拖拽 | 支持用户拖拽到屏幕左右两侧，松手后自动吸附到最近的边缘 |

---

## 交互

| 行为 | 说明 |
|---|---|
| 点击 | 触发主操作（如打开客服、扫码等） |
| 长按/拖拽 | 可拖拽移动位置，松手吸附到左/右边缘 |
| 关闭 | 点击关闭按钮后隐藏，当前会话内不再显示 |
| 页面滚动 | 位置固定不动，不随页面滚动 |

---

## Do Not Infer

- 不要让浮动按钮遮挡页面核心操作区域。
- 不要同时展示多个浮动按钮 — 最多一个。
- 不要让浮动按钮尺寸偏离 40×40px。
- 不要省略投影 — 投影是浮动按钮与页面内容的层级区分。
- 不要让浮动按钮在页面中间 — 必须贴边展示。
